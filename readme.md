# Valutics AI Systems Engineer Assignment - Budget Tracker

## Overview
This is a console-based budget tracker application built in Python. It allows users to manage their income and expenses through a simple command-line interface. The application uses object-oriented programming (OOP) and handles data persistence by saving transactions to a file.

## Features
-   *Add a Transaction*: Enter details for new income or expense entries.
-   *List All Transactions*: View all transactions in a readable format.
-   *Filter Transactions*: Filter transactions by type (income/expense) or category.
-   *View Summary*: See total income, total expenses, and the net balance.
-   *Save & Load*: All data is automatically saved to transactions.txt and loaded on startup.

## How to Run
1.  *Clone the repository:*
    bash
    git clone [Your-GitHub-Repo-URL]
    cd [Your-GitHub-Repo-URL-folder]
    
2.  *Run the application:*
    bash
    python budget_tracker.py
    
3.  Follow the menu prompts to interact with the budget tracker.
